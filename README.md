# Todo List

This is a React-based Todo List application built as part of an introductory web development assignment.

## Description

A simple, interactive application that renders a list of tasks. This project serves as an introduction to React fundamentals, including component structure, JSX syntax, and rendering lists from arrays using the map function.

## Installation

*Ensure you have **[Node.js](https://nodejs.org/)** installed on your machine.*

1. Clone the repository to your local environment:
```bash
git clone <your-repo-url>
```

2. Navigate into the project directory:
```bash
cd todo-list
```

3. Install the project dependencies:
```bash
npm install
```

## How to Run the Development Server

After installing the dependencies, you can start the development server by running:
```bash
npm run dev
```

Once the server is running, navigate to the URL provided in your terminal (usually `http://localhost:5173`) to view the application in your browser.
